# Avatar Placeholder

Letakkan file avatar Anda di sini dengan nama `avatar.jpg` atau `avatar.png`.

Untuk sementara, Hero component akan menampilkan initial name sebagai placeholder jika avatar tidak tersedia.

Rekomendasi:
- Format: JPG atau PNG
- Ukuran: 400x400px atau lebih (square/persegi)
- File size: < 200KB untuk performa optimal
